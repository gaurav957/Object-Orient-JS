Features:-
Encapsulation:- Object data is inside an Object package
Object Package:- Its called a nameSpace(function in case of constructer function).
Abstraction:- Creating something simple from a more complex one.
Object Instances:- Contains detailed functionalities inside a class. When created a constructer is called and its called instantiation.
Inheritance:-
Prototypes: Javascript objects inherits features from one another.
